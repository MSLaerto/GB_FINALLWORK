    a. Создание и заполнение файлов:
    cat > "Домашние животные" << EOF
    собаки
    кошки
    хомяки
    EOF

    cat > "Вьючные животные" << EOF
    Лошади
    верблюды
    ослы
    EOF
    

    b. Объединение файлов:
    cat "Домашние животные" "Вьючные животные" > "Животные"
    

    c. Просмотр содержимого файла:
    cat "Животные"
    

    d. Переименование файла:
    mv "Животные" "Друзья человека"
    

2. Создание директории и перемещение файла:
    
    mkdir pets
    mv "Друзья человека" pets/
    

3. Подключение дополнительного репозитория MySQL и установка пакета:

    sudo add-apt-repository 'deb http://repo.mysql.com/apt/ubuntu/ bionic mysql-5.7'
    sudo apt update
    sudo apt install mysql-server
    

4. Установка и удаление deb-пакета:
    - Установка:

    sudo dpkg -i путь_к_файлу.deb
    
    - Удаление:

    sudo dpkg -r название_пакета
